Senior Software Engineer

Fortune 500 Client Remote (US) Contract · Full-Time $120,000–$160,000/yr

Job Overview

Experienced Senior Software Engineer for cloud-native applications at a Fortune 500 client. Work on cutting-edge AI-powered solutions for enterprise clients. Remote position open to US-based candidates.

Schedule: Remote — US time zones

Responsibilities

  • Design and build scalable cloud-native applications using React, Node.js, and Python
  • Collaborate with Product and Design to deliver high-quality features
  • Participate in architecture decisions and technical roadmap planning
  • Conduct code reviews and mentor junior engineers
  • Deploy and maintain applications on AWS infrastructure
  • Work with cross-functional teams across multiple time zones

Requirements & Qualifications

  • 5+ years of full-stack software engineering experience
  • Expert-level proficiency in React and Node.js
  • Strong experience with AWS (EC2, Lambda, S3, RDS)
  • Python proficiency for backend services and automation
  • Experience with cloud-native architecture patterns
  • Excellent communication and collaboration skills

Education

Bachelor's degree in Computer Science or related field, or equivalent experience.

Skills & Certifications

React Node.js AWS Python Cloud Architecture TypeScript
