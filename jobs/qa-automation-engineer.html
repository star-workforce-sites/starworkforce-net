QA Automation Engineer

Nearshore Client Nearshore (Canada) Contract · Full-Time $85,000–$110,000/yr
Apply Now →

Job Overview

QA Automation Engineer to create and maintain automated testing frameworks for web and mobile applications at a nearshore client in Canada. Work with cross-functional teams to ensure quality across the full development lifecycle.

Schedule: Nearshore — Canada

Responsibilities

  • Design, build, and maintain automated test frameworks using Selenium, Cypress, and Jest
  • Develop end-to-end, integration, and unit test suites
  • Collaborate with developers and product managers to define test strategies
  • Identify, document, and track software defects through resolution
  • Participate in sprint planning and provide QA estimates
  • Maintain CI/CD pipeline integration for automated test execution

Requirements & Qualifications

  • 3+ years of QA automation engineering experience
  • Proficiency in Selenium WebDriver for browser automation
  • Experience with Cypress for modern web application testing
  • Jest or similar JavaScript testing framework experience
  • Strong JavaScript/TypeScript skills
  • Experience integrating test automation into CI/CD pipelines

Education

Bachelor's degree in Computer Science, Engineering, or equivalent experience.

Skills & Certifications

Selenium Cypress Jest JavaScript TypeScript CI/CD Test Automation
